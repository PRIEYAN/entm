# Pinned deps for the arm64 serving image (Containerfile).
# All must ship prebuilt aarch64 wheels. No torch. No IndicTransToolkit.
#
# transformers is 4.57.6 to match requirements/pi.txt and the version the model
# was converted and verified with (ctranslate2 4.8.1 needs >=4.56).
#
# Unlike requirements/pi.txt these stay EXACT-pinned, deliberately: the image
# fixes the interpreter at python:3.11-slim, where every version below has a
# prebuilt aarch64 wheel, so pinning buys reproducible builds with no source-
# build risk. pi.txt has to float because bare-metal Python varies by board
# image (DietPi ships 3.13, where some of these pins have no wheel).

ctranslate2==4.8.1
transformers==4.57.6
# 0.2.1, not 0.2.0: 0.2.0 has no aarch64 wheel for Python 3.13 and would
# trigger a source build (needs cmake + pkg-config) inside the image.
