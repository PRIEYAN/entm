# ON THE JETSON NANO (Python 3.8): use requirements/jetson.txt instead of this
# file. pi.txt's exact pins (ctranslate2==4.8.1, transformers==4.57.6) have no

# resolve on 3.8 and already includes faster-whisper + the mic stack:
#
#     pip install -r requirements/jetson.txt
#
# This file targets a modern Python (3.9+) box and keeps pi.txt's exact pins.
#
# ⚠️ GPU note: the plain PyPI `ctranslate2` wheel is CPU-only. To run STT AND
# translation on the Jetson GPU you need a CUDA-enabled CT2 build for the Nano
# (CUDA 10.2 / sm_53) — install it from an NVIDIA l4t container or a source
# build (see nvidia/NvidiaRun.md + NvidiaPlan.md §3.2). If that build isn't
# present, engine.py falls back to CPU automatically and the pipeline still runs.
